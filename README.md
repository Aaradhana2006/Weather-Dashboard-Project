# 🌤️ Weather Dashboard

A modern and responsive Weather Dashboard built using HTML, CSS, and JavaScript. This application fetches real-time weather information from the OpenWeather API and displays key weather metrics for any searched city.

## 🚀 Features

* Search weather by city name
* Real-time weather data using OpenWeather API
* Temperature, Humidity, Wind Speed, and Visibility
* Weather condition with dynamic icon
* Async/Await and Fetch API implementation
* Error handling for invalid city names
* Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* OpenWeather API
* JSON
